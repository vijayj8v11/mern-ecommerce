### Recommended Screenshot Dimensions

- **Desktop Screenshots**: 1280×720px (16:9 aspect ratio)
- **Mobile Screenshots**: 375×812px (iPhone dimensions)
- **File Format**: PNG (lossless) or optimized JPG
- **File Size**: Aim for < 500KB per image

### Tools to Take Screenshots

1. Windows Built-in: `Print Screen` → Paste in Paint
2. Greenshot - Free screenshot tool
3. Snagit - Professional screenshot tool
4. Chrome DevTools - Built-in device emulation
5. VS Code Screenshots Extension

### Hosting External Images

If images are large, consider hosting them externally:
- GitHub Issues (upload when creating issue)
- Imgur
- Cloudinary
- AWS S3

Then reference them in README with the full URL.
